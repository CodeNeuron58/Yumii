## What is Yumii?

Yumii is an open-source AI companion that runs on your own machine. She
listens through your microphone, thinks with the LLM provider you choose,
answers out loud in a **local voice**, and shows up as a small floating orb —
always there, never in the way.

Three promises:

* **Voice-first.** Real conversation, not a chat box. Interrupt her
  mid-sentence and she stops and listens.
* **Private by design.** Conversations, memory, and API keys are local files
  on your disk. The only network traffic goes to the providers you configure.
* **She asks before she acts.** Every tool that touches the outside world
  goes through an Approve / Deny prompt first.

## What she can do today

| | |
|---|---|
| **Talk in real time** | Barge-in interruption, ~1s reply latency on an ordinary CPU |
| **Six personalities** | caring · tsundere · genki · kuudere · yandere · dandere — switch by asking |
| **Remember you** | Learns facts, searches every past conversation, knows *when* things happened |
| **Do things** | Web search built in; Gmail, Calendar, Notion via Composio — all permission-gated |
| **Stay inspectable** | A dashboard shows every conversation, every fact, every connected tool |
| **Speak locally** | Kokoro-82M synthesizes her voice on your CPU — free, offline, no key |
