## рџҺ›пёЏ Interactive Shell Commands

When you launch the console using `yumii` or `uv run yumii`, you enter a rich REPL command-line interface.

The following command shortcuts are registered:

| Shell Command | Alias Command | Description |
| :--- | :--- | :--- |
| **`/wake`** | `wake`, `wake-up` | **Wake Yumii up** вЂ” starts the FastAPI backend on port `8000`, routes audio, and automatically opens your web browser. |
| **`/chat`** | `chat`, `new` | **Chat picker** вЂ” TUI to resume a saved session, start a new one, or delete a session. |
| **`/resume`** | `resume`, `resume-chat` | **Resume last session** вЂ” opens the most recently active session in the browser. |
| **`/sessions`** | `sessions` | **List sessions** вЂ” prints a Rich table of all saved chat sessions. |
| **`/memory`** | `memory` | **Memory manager** вЂ” TUI to browse, edit, or delete long-term user facts. |
| **`/forget`** | `forget` | **Wipe memory** вЂ” erase all stored facts (sessions are kept). |
| **`/name <name>`** | `rename` | **Rename session** вЂ” works in the browser (the REPL guides you there). |
| **`/config`** | `configure`, `settings` | **Configure Senses** вЂ” launches the step-by-step Attunement wizard to set up Groq, OpenAI, ElevenLabs, and Whisper configurations. |
| **`/personality`** | `personality` | **Select Personality** вЂ” dynamically switches Yumii's active personality prompt and loaded visual emotion state. |
| **`/vision`** | `story`, `roadmap` | **Vision & Story** вЂ” loads Yumii's project origin, development roadmap, and feature visions inside the terminal viewport. |
| **`/help`** | `h`, `?` | Displays the stylized Command help matrix inline. |
| **`/exit`** | `quit`, `q` | Exits the dashboard gracefully, shutting down background tasks and returning to the system terminal. |

## рџљЂ Direct Terminal Shortcuts

You can bypass the interactive console REPL and execute commands directly from your standard terminal prompt:

```bash
# Wake Yumii up immediately
uv run yumii wake-up

# Configure settings directly
uv run yumii attune

# Resume your most recent chat
uv run yumii resume-chat

# Wipe long-term memory (facts only, sessions stay)
uv run yumii forget

# Hard-delete a single session by id
uv run yumii delete-session <uuid>

# Check the package version
uv run yumii --version
```

Proceed to the [Configuration Schema Reference](/ops/configuration-schemas) page to see config JSON layouts!
