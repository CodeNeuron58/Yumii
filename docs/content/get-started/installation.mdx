## Windows — the installer (recommended)

The installer bundles everything: the backend, the speech engine, and Yumii's
local voice model. **No Python, no dependencies, and the first run works
offline** — the only thing you'll ever need to enter is one API key for her
thinking.

<StepsList>
  <Step icon="download" title="Download">
    Grab the latest <code>Yumii_…_x64-setup.exe</code> from the{' '}
    <a href="https://github.com/CodeNeuron58/Yumii/releases">Releases page</a>.
    It's around 450&nbsp;MB — the voice model is inside, which is why the first
    run needs no download.
  </Step>
  <Step icon="windows" title="Run it (and get past SmartScreen)">
    Yumii is a new open-source project and isn't code-signed yet, so Windows
    will likely show a blue <strong>"Windows protected your PC"</strong> screen.
    This is what Windows shows for <em>any</em> unsigned installer — click{' '}
    <strong>More info → Run anyway</strong>.
  </Step>
  <Step icon="play" title="Launch">
    Start Yumii from the Start menu. A small floating orb appears and says
    "Waking Yumii up…" while the backend boots (10–20 seconds on first launch).
  </Step>
</StepsList>

<Callout type="info" title="Is the SmartScreen warning safe?">
Yes. Code-signing certificates cost hundreds of dollars a year, which young
open-source projects usually skip. The source code is public and the installer
is built by a public GitHub Actions workflow you can inspect. If you'd rather
not click through the warning, run from source instead.
</Callout>

An `.msi` package is also attached to each release for managed or enterprise
installs.

## macOS & Linux — from source

There's no installer for macOS/Linux yet, but Yumii runs from source on both.
You need **Python 3.12+** and [`uv`](https://docs.astral.sh/uv/):

```bash
# install uv if you don't have it
curl -LsSf https://astral.sh/uv/install.sh | sh

git clone https://github.com/CodeNeuron58/Yumii.git
cd Yumii
uv sync
uv run yumii
```

`uv run yumii` opens the CLI; pick **Wake Yumii Up** and your browser opens to
the orb. On first run, her voice model (~350&nbsp;MB) downloads automatically to
`~/.yumii/models/`.

<Callout type="warning" title="Use uv, not pip">
The project pins <code>torch</code>-adjacent packages to CPU-only builds that
plain <code>pip</code> doesn't resolve correctly. Always <code>uv sync</code>.
</Callout>

## The desktop app from source (developers)

The frameless orb window is a Tauri (Rust) app. To run it from source you also
need Rust and — on Windows — the MSVC C++ Build Tools:

```bash
cd desktop
npx @tauri-apps/cli dev
```

This launches the orb window *and* starts the Python backend for you. Details
in [Running from Source](/development/from-source).

## What gets installed where

| Thing | Location |
|---|---|
| The app (installer) | Standard Program Files install, with a Start-menu entry |
| Your settings & API keys | `~/.yumii/` (survives uninstall/reinstall) |
| Conversations & memory | `~/.yumii/memory/` |
| Voice models (source installs) | `~/.yumii/models/` |

Uninstalling the app never deletes `~/.yumii` — your memory, keys, and
conversations are yours and stay put. Delete that folder yourself if you want
a true clean slate. Full layout in [File Locations](/reference/files).
