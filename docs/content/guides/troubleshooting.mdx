# Troubleshooting

If you encounter issues while running Yumi, check this guide for common solutions.

## Common Issues

### API Key Errors

**Symptom:** You receive an authentication error or a message that an API key is missing.

**Solution:**
Yumi stores API keys in your **OS keychain**, not environment variables. Re-run the setup wizard:
```bash
uv run yumi        # from git clone
# or just:
yumi               # if installed via the one-liner
```
Then go to **Configure Senses** to re-enter your API keys.

### Audio Device Not Found

**Symptom:** Yumi cannot hear you or fails to initialize the microphone.

**Solution:**
Make sure your microphone is properly connected and recognized by your operating system. You might need to configure the default audio input device.

## Getting More Help

If you're still stuck, open an issue on the [GitHub repository](https://github.com/CodeNeuron58/Yumi/issues).
