# Troubleshooting

If you encounter issues while running Yumii, check this guide for common solutions.
</ContentCard>

<ContentCard>
## Common Issues

### API Key Errors

**Symptom:** You receive an authentication error or a message that an API key is missing.

**Solution:**
Yumii stores API keys in your **OS keychain**, not environment variables. Re-run the setup wizard:
```bash
uv run yumii        # from git clone
# or just:
yumii               # if installed via the one-liner
```
Then go to **Configure Senses** to re-enter your API keys.

### Audio Device Not Found

**Symptom:** Yumii cannot hear you or fails to initialize the microphone.

**Solution:**
Make sure your microphone is properly connected and recognized by your operating system. You might need to configure the default audio input device.
</ContentCard>
