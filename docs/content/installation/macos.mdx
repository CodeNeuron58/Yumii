## ⚡ One-Line Quick Install

Open a standard Terminal window and run the automated bash installation script:

```bash
curl -LsSf https://raw.githubusercontent.com/CodeNeuron58/Yumii/master/install.sh | sh
```

### What this script does:
1.  Installs the `uv` package manager securely via the official Astral installer.
2.  Runs `uv tool install git+https://github.com/CodeNeuron58/Yumii.git` — installs Yumii as a uv-managed tool (no manual clone, no separate venv). The CPU-only PyTorch wheel pin in `pyproject.toml` is honored automatically.
3.  The `yumii` command becomes globally available — `uv tool install` puts it on `~/.local/bin` for you, no shell alias needed.

### Prerequisites

`install.sh` assumes a standard `curl` + POSIX shell environment and a working network connection. It does not install Homebrew, Python, or system audio libraries — see the audio section below for those.

## 🛠️ macOS Audio Permissions & Dependencies

Because of macOS's strict sandbox security, a couple of steps are required to ensure audio capture and keyring storage work flawlessly:

### 1. Terminal Microphone Permissions
If you run Yumii through a terminal emulator (like Terminal, iTerm2, or VS Code), that app must be granted permission to access your microphone.
1.  Go to **System Settings > Privacy & Security > Microphone**.
2.  Ensure your terminal app (e.g. `Terminal` or `iTerm2`) is toggled **ON**.
3.  If Yumii still fails to hear you, try running her from a clean terminal window so macOS can trigger the native permission prompt.

### 2. Apple Silicon (M1/M2/M3) Optimization
Apple Silicon chips include hardware matrix coprocessors (AMX) that accelerate local speech recognition.
*   **Performance**: If using **Local Whisper** on Apple Silicon, execution is extremely fast and light.
*   **Keyring Security**: Yumii securely commits your API keys to the **macOS Keychain Access** utility. The first time she queries the keychain, macOS will display a secure system dialog asking for your password or TouchID to authorize access. Select **Always Allow**.
